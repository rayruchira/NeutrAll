Workplace diversity is important not only because it brings benefits in business spheres such as brand management , customer engagement, and CSR, but also because it creates added value for the organization by attracting diverse innovation driving professionals.

WHAT DO WE DO?

Virtual HR

NeutrAll helps to track inclusion at any given organization. Employees provided with a platform to express themselves and their concerns regarding how acknowledged they feel. We protect identities so that employees feel safe.

Auth0 Authentication

Rapidly integrating authorization for our web application using Auth0 API, which provides an extra layer of security by enabling Multi-factor Authentication. Administrators can manage users, provisioning blocking and deleting users going against company protocol.

Personality Insight API

Analysisng social media presence (such as, Instagram) to evaluate inclusiveness and preferences of the organisation by integrating Personality Insight API by IBM.

Interactive Charts

We provide interactive charts using google sheets API to show real-time changes according to company employees list. Statistics according to user identities are shown and a free platform is provided for easy change or preference and perspectives.
